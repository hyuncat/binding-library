## Binding Library

Original code by Dan Redeker, adapted by Sarah Hong.

To run, clone the repository and run:
```shell
pip install -r requirements.txt
```

An overview of the functions are given in `binding_library.ipynb`. Otherwise, a terse sample run is given in `run.ipynb`.
